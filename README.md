 .dotfiles
======
### personal dotfiles for bspwm, managed with Dotbot.


#### installing:

```bash
cd && git clone https://github.com/iofq/.dotfiles && cd .dotfiles && ./install
```

#### alternatively, run in docker:
```bash
docker run --rm -it iofq/dots:latest
```
